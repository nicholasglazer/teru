# teru Roadmap

Internal todo list. Version convention: 0.1.x = patch (small features/fixes), 0.x.0 = minor (major features).

---

## 0.1.2 — Wire existing dead code + quick wins

- [ ] Wire HookHandler into main event loop (currently dead code — parse 5 events, expand to 25)
- [ ] Wire prefix_key config (currently parsed but hardcoded to Ctrl+Space)
- [ ] Pane zoom (Ctrl+Space,z — toggle active pane to monocle and back)
- [ ] Pane resize (Ctrl+Space,H/J/K/L — shrink/grow by 1 cell)
- [ ] Status bar (bottom row: workspace indicator, pane count, agent count, clock)
- [ ] OSC 8 hyperlinks (clickable links — ls, eza, fd, bat emit these)
- [ ] Synchronized output (DEC mode 2026 — prevents tearing)

## 0.1.3 — Agent-aware rendering

- [ ] Agent-aware pane borders (cyan=running, green=done, red=failed, dim=idle)
- [ ] Tool activity indicator in border ("Reading file.zig...", "Running tests...")
- [ ] Progress bars in pane borders (OSC 9999 agent:status progress=0.6)
- [ ] Auto-register MCP server when Claude Code detected (CLAUDECODE env var)
- [ ] Notification routing (OSC 9 → desktop notification + status bar)

## 0.1.4 — Wayland parity + clipboard

- [ ] Wayland keyboard: use compositor keymap FD via xkb_keymap_new_from_fd()
- [ ] Copy mode (vi-like scrollback navigation: Ctrl+Space,[ enters, hjkl/v/y)
- [ ] Custom keybindings (config file key=action mapping beyond prefix keys)

## 0.1.5 — Terminal basics

- [ ] Kitty keyboard protocol (disambiguate press/release/repeat, modifier encoding)
- [ ] Shell integration scripts (bash/zsh/fish auto-inject OSC 133 emitters)
- [ ] Scrollback search improvement (persist query, highlight count, next/prev match)

## 0.2.0 — Detach/attach daemon

- [ ] Daemon mode (teru-server background process owns PTYs)
- [ ] teru attach (client connects to running daemon, renders to its own window)
- [ ] teru detach (disconnect client, daemon keeps PTYs alive)
- [ ] teru list (show running sessions)
- [ ] Session persistence across terminal restarts
- [ ] Named sessions (teru new -s work, teru attach work)

## 0.3.0 — Graphics + Unicode

- [ ] Kitty graphics protocol (bitmap display, streaming, placement)
- [ ] Sixel graphics (legacy image protocol)
- [ ] Full Unicode: emoji via COLR/CPAL tables
- [ ] Full Unicode: CJK double-width characters (wcwidth)
- [ ] Full Unicode: grapheme cluster segmentation
- [ ] Font fallback chains (try Hack → Noto → system fonts)
- [ ] Ligatures (programming font ligatures — Fira Code, JetBrains Mono)

## 0.4.0 — Plugin system

- [ ] WASM plugin runtime (wasmi or wasmtime, sandboxed)
- [ ] Plugin API: hooks (on_key, on_output, on_resize, on_agent_event)
- [ ] Plugin API: render (custom status bar widgets, overlays)
- [ ] Plugin API: tools (register MCP tools from plugins)
- [ ] Plugin manager (install/update/remove from registry)
- [ ] Lua scripting bridge (Lua 5.4, for config/keybind scripting)

## 0.5.0 — AI-first features (the differentiator)

- [ ] Context compaction gauge (status bar shows Claude Code context usage %)
- [ ] Agent cost tracking (accumulate tokens from hook metadata, show per-agent cost)
- [ ] Cross-pane approval (permission_prompt hook → inline yes/no widget without switching panes)
- [ ] Auto-workspace for teams (spawn_agent with group → dedicated workspace)
- [ ] Process graph visualization (Ctrl+Space,g — DAG view, click to navigate)
- [ ] Agent replay (rewind session using scrollback + event timestamps)
- [ ] Agent output cross-reference (highlight source pane region when another agent reads via MCP)

## 0.6.0 — macOS

- [ ] Fix AppKit backend (@Type removal in Zig 0.16 — explicit function pointer casts)
- [ ] PTY via forkpty() from util.h
- [ ] Keyboard via Carbon UCKeyTranslate (native, no xkbcommon dep)
- [ ] Clipboard via pbcopy/pbpaste
- [ ] Homebrew tap (nicholasglazer/homebrew-teru)
- [ ] CI: GitHub Actions macOS runner
- [ ] Retina/HiDPI support (NSScreen backingScaleFactor)

## 0.7.0 — Windows

- [ ] ConPTY (CreatePseudoConsole, Win10 1809+)
- [ ] Win32 keyboard translation (MapVirtualKeyW + ToUnicode)
- [ ] Clipboard (OpenClipboard + SetClipboardData)
- [ ] Config paths (%APPDATA%\teru\teru.conf)
- [ ] Named pipes for MCP/PaneBackend sockets
- [ ] Scoop bucket + WinGet manifest
- [ ] CI: GitHub Actions Windows runner

## 0.8.0 — SDK integration

- [ ] Claude Code SDK embedding (optional Node.js/Python process for terminal-native AI chat)
- [ ] Headless session spawning (claude -p --output-format stream-json, NDJSON parsing)
- [ ] Session recording with semantic markup (commands, agent events, timing)
- [ ] Session replay (searchable, shareable recordings)

## Future / unversioned
